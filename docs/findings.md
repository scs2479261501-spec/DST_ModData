- `常青树` 不应直接写成“仍在积极维护”。虽然它们同时具备高订阅和较长维护跨度，但平均距离今天已经有 `1,515.80` 天，更准确的描述是“长寿且高 adoption 的 mod”。
- `爆款弃坑` 是最陈旧的象限。它的平均距今天数达到 `1,929.97` 天，说明这类 mod 虽然吃到过高订阅红利，但后续基本没有持续维护。
- `用爱发电` 的订阅量很低，但维护跨度明显长于 `爆款弃坑` 和 `沉默消亡`。这一类更适合在报告里解释为“小众但持续投入”，而不是“失败作品”。
- `沉默消亡` 数量最多，但没有夸张到 `80%` 这种会直接压垮分析的程度。因此当前结果可以直接进入可视化阶段。

- `interface`、`environment`、`utility` 这些标签，在“高订阅 + 更长维护跨度”上明显优于整体均值，后续可以优先作为 Dashboard 的重点观察对象。
- `all_clients_require_mod` 虽然是最大的标签族，但它的四象限结构和整体市场接近，本身不是很强的解释变量，更适合作为背景标签而不是结论中心。

## 模块三：作者生产力分析

分析口径：
- 数据批次：`api_full_20260319`
- 作者主键使用 API 返回的 `creator` 字段
- `creator` 为空的异常记录不进入作者分析
- 作者级指标包括：mod 数、总订阅、平均订阅、中位订阅、平均好评率、平均维护时长、标签覆盖广度

作者总体情况：
- 参与分析的作者数：`9,626`
- 作者总 mod 数：`22,919`
- 人均 mod 数：`2.38`
- 最高产作者的 mod 数：`127`

头部集中度：
- 前 `1%` 作者共 `97` 人，占全部作者总订阅量的 `63.99%`
- 前 `10` 名作者占全部作者总订阅量的 `27.35%`
- 这说明 DST mod 生态的作者侧非常头部化，不是“均匀分布”的长尾市场

头部作者样本：
- 订阅总量第一的作者：`76561198025931302`
  - mod 数：`10`
  - 总订阅：`27,176,956`
  - 平均订阅：`2,717,695.60`
  - 平均好评率：`0.973160`
- 高产作者不一定是头部订阅作者。最高产作者 `76561199116111540` 有 `127` 个 mod，但总订阅只有 `39,723`，平均订阅 `312.78`。
- 同时也存在“既高产又能打”的作者，例如 `76561198204009267` 拥有 `62` 个 mod，总订阅 `3,759,803`，平均订阅 `60,641.98`。

高产是否高质：
- `mod_count` 和 `avg_subscriptions` 的 Pearson 相关系数：`0.014409`
- `mod_count` 和 `avg_positive_rate` 的 Pearson 相关系数：`0.007082`
- 这两个值都非常接近 `0`，说明从线性关系上看，“高产作者更高质”并不成立，“高产作者更低质”也同样不成立。

按产量分层看作者表现：
- `1` 个 mod 的作者：`6,092` 人，作者平均单作订阅 `7,832.20`，平均好评率 `0.849807`
- `2-3` 个 mod 的作者：`2,183` 人，作者平均单作订阅 `11,650.56`，平均好评率 `0.863371`
- `4-9` 个 mod 的作者：`1,029` 人，作者平均单作订阅 `10,030.00`，平均好评率 `0.859552`
- `10+` 个 mod 的作者：`322` 人，作者平均单作订阅 `21,175.19`，平均好评率 `0.864324`

模块三的核心判断：
- 作者生态高度头部集中，前 `1%` 作者吃掉了接近三分之二的总订阅。这是一个非常典型的平台型内容生态特征。
- “高产”本身不是稳定的质量保证。线性相关几乎为零，说明单靠产量推不出作者一定更强。
- 但高产作者群体的平均单作订阅确实更高，尤其 `10+` 组明显高于单作作者组。这更像是“高产作者里更容易出现成熟创作者”，而不是“每多做一个 mod 就会线性提升质量”。
- 平均好评率在各产量层之间差距很小，说明高产并没有明显带来口碑崩塌，也没有带来显著口碑溢价。
- 因此，作者分析更适合写成“生态头部集中 + 产量与表现弱相关”，而不是简单下结论说“高产一定高质”或“高产一定低质”。

对可视化的直接建议：
- 第一张图做作者总订阅 Lorenz 式集中度展示，或者至少做 Top 1% / Top 10 的占比卡片。
- 第二张图可以做作者 `mod_count` vs `avg_subscriptions` 散点图，再用点颜色表示 `avg_positive_rate`。
- 第三张图可以做产量分层条形图，直接比较 `1`、`2-3`、`4-9`、`10+` 四档作者的平均单作订阅与平均好评率。

- 这批评论样本受到 `429` 限流和英文分词口径限制，结论适合用于方向性判断，不应当包装成全量用户语言画像。
